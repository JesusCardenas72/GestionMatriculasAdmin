# Gestion de Matriculas - Admin

App de escritorio (Electron + React + TS + Vite + Tailwind) para administrar solicitudes de matricula almacenadas en Dataverse via flows de Power Automate.

Ver `g:/Dev/PLAN-GestionMatriculasAdmin.md` para el plan completo.

## Desarrollo

```
npm install
npm run dev
```

## Build

```
npm run build
npm run dist
```
